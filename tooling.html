Libraries & developer tooling

A published library and the developer-facing tools I build for my own daily use.

Ax AxisDB ↗ A tiny embedded document database for Python, addressing JSON documents by N-dimensional coordinate keys. Published to PyPI: pip install axisdb. Lo Locus A native Windows task board that mirrors coding-agent tool calls onto a live board in real time. Cf CommandFixer PowerShell command autocorrect for mistyped commands. Ca Coin Analysis A numismatic coin analyser using a hybrid of OCR and a scikit-learn classifier.
